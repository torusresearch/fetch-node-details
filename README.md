# fetch-node-details

## Introduction

This monorepo contains various packages to fetch node config of various web3auth's networks.

It has following pacakges:-

- @toruslabs/constants
- @toruslabs/fnd-base
- @toruslabs/fetch-node-details
- @toruslabs/fnd-server


## Installation

- npm i

## Run server

- npm run dev

## Run test cases

- npm run test:ci


## Build packages

- npm run build
